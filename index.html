Cookbook

V rámci předmětu ITU jsem vytvořil full-stack aplikaci pro filtrování receptů. Zde jsem měl zodpovědnost za kompletní backend (návrh, API, databáze) a práci na frontendu jsme si rozdělili v týmu.

Firestamps Recorder

Soubor scriptů pro stříhání videí hasičských útoků. Jsou tu scripty pro stahování videí z YouTube, script pro úpravu kodeků a a spojování videí, GUI pro ukládání časových značek, samotné stříhání videí a přidávání overlaye se značkami.

Firestamps Scraper

Script pro stahování výsledů z hasičských soutěží z webu firesport.eu. Tento script má workflow na GitHubu a spouští se každý den. Tato data jsou uložena v Supabase a použita na webu, který tato data zobrazuje a umožňuje jejich filtrování. (WIP)

Tato osobní webová stránka

Tato moderní responzivní webová stránka vytvořená pomocí HTML, CSS a JavaScriptu. I když se všude používají frameworky, tento projekt mi pomohl pochopit základy html, css a javascriptu, které jsem pod vrstvou frameworků často tolik neviděl.

Automatizace pro firmu Heralt

Automatizace procesů v google sheets pro restauraci Heralt. Tento projekt zahrnuje skripty pro automatizaci správy objednávek, vytváření faktur, přepisu objednávek do faktur a jejich ukládání, a další. Tento projekt mě hodně naučil o uživatelské přívětivosti, práci s velkým množstvím dat, práci s API a spolehlivosti.
